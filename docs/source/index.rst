maya_data_lab
=============

Research and prototype project exploring Git-friendly, deterministic data formats
for Maya asset data.

This project focuses on:
- Externalizing Maya data (UVs, skin weights, etc.)
- Building clean Python API wrappers around Maya nodes
- Enabling reproducible, versionable asset builds
